# AI Portfolio – Shubham Pandey

An interactive AI/ML portfolio built using React, Vite, Three.js, and React Three Fiber.

The website showcases my projects, skills, and experience through an immersive neural-network-inspired interface featuring 3D visualizations, brain morphing animations, project clusters, and AI-focused design.

---

## Features

### Interactive Neural Sphere

* Procedurally generated neural network visualization
* User-controlled rotation
* Smooth transitions and animations
* Dynamic node connections

### Brain Morph Animation

* Neural sphere morphs into a brain structure
* Animated transition between portfolio states
* Custom particle movement and interpolation

### AI Project Showcase

* Computer Vision projects
* Agentic AI systems
* NLP applications
* Recommendation Systems

### Skill Network Visualization

* Interactive neural-style skill graph
* Categorized technical expertise
* Dynamic connections between domains

### Responsive Portfolio Sections

* Home
* About
* Projects
* Skills
* Contact

## Installation

Clone the repository:

```bash
git clone https://github.com/Shubhampandey1git/portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
